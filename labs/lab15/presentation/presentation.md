# Основные этапы работы 
- Скачиваем файл hugo_extended_Linux_64bit.tar.gz 
- Разархивируем данный файл
- Создание папки bin
- Создаем новый репозиторий
- Клонируем его
- Запускаем сервер
- Коммитнем наш репозиторий и нашу папку, создадим файл
